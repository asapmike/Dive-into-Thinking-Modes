# Think Out Loud or Think Inside?

Static GitHub Pages prototype for a CSE 151B communication-track artifact.

The page is a deterministic interactive explainer. It uses scripted puzzle
responses to compare direct answering, visible chain-of-thought, hidden
reasoning, and latent reasoning as a research preview. It does not call a live
model or claim to expose real model internals.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.

## Deploy with GitHub Pages

After pushing this folder to GitHub, enable Pages for the repository from the
`main` branch and the repository root.
